# Brand Guide — 확정 브랜드 정체성

이 문서는 이 프로젝트의 **현재 확정된** 브랜드 정체성을 담는다. Figma "Brand Guide" 페이지(비주얼 원본)의 텍스트 미러이자, 다른 모든 에이전트가 브랜드 값을 참조하는 소스 오브 트루스다. brand-designer가 브랜드 방향을 정식화·변경할 때마다 이 파일을 **덮어써서** 항상 최신 상태로 유지한다(로그가 아니다 — 지난 결정은 git 히스토리에 남는다).

## 레이아웃 관례 (Layout Convention — B-2)

확정된 2차 레이아웃 시안 **B-2**("브랜드 컨셉 Concepts" 페이지 `34:2` 내 프레임 `62:6`, 좌측 사이드바 내비 + 테이블형 리스트)를 직접 다시 보고 관찰한 실제 관례다. **컬러 hex·폰트명만으로는 이 시안이 "좋아 보이는" 이유를 재현할 수 없다** — 아래 간격/보더/라운드/타이포 위계까지 후속 화면(SCREENS)에서 그대로 따라야 한다.

- **구조 (Structure)**: 사이드바(고정폭 약 212px, 주조색 teal 풀블리드) + 테이블형 리스트 본문. 사이드바는 위에서부터 브랜드 로고(원형 심볼+워드마크) → "CATEGORY" 내비게이션(항목마다 우측에 pill형 카운트 배지) → "카테고리 관리"(인라인 수정/삭제) → 새 카테고리 추가 폼 순으로 쌓는다. 본문은 상단바(타이틀+사용자명+로그아웃) → 검색행 → 다중 입력 필드를 한 박스로 묶은 "추가" 행 → 리스트 헤더(총 건수) → 테이블(헤더+행) → 상태 메시지 배너 순으로 배치한다.
- **보더 위계 (Border Weight)**: 그림자(elevation)는 전혀 쓰지 않는다 — 잉크 보더(`#1A1A1A`) 두께로만 구조적 중요도를 표현한다. **3px** = 화면 최외곽 프레임 · **2.5px** = 그룹 컨테이너(테이블 전체, 여러 입력 필드를 한 번에 묶는 "추가" 행 박스) · **1.5px** = 개별 입력창·버튼 · **1px** = 행 내부 소형 액션 버튼(수정/삭제). 테이블 행 사이 구분선만 예외로 잉크가 아닌 연한 회색(`#E0E0E0`) 헤어라인 1px을 써서, 고밀도 리스트에서 시각적 소음을 줄인다.
- **라운드 스케일 (Corner Radius)**: 16px 화면 최외곽 프레임 · 10px 테이블/그룹 박스 · 8px 버튼·입력창·내비게이션 아이템·상태 배너 · 5px 행 내부 소형 액션 버튼 · 999px(pill) 카운트 배지. 이 5단계 스케일 밖의 임의의 radius 값은 추가하지 않는다.
- **간격 리듬 (Spacing Rhythm)**: 4px 그리드 기반. 사이드바 내부 블록 간 gap 16px, 본문 블록 간 gap 14px, 테이블 행/헤더 padding 12×9px, 입력창 padding 10×7px, 일반 버튼 padding 12×7px, 행 내부 소형 버튼 padding 8×4px. 본문 컨테이너 padding은 20px로 절제해 고밀도 정보 배치를 우선한다 — 여백을 넉넉히 주는 미니멀 스타일과는 다른 방향이다.
- **정보 밀도 타이포 (Density Typography)**: 화면 타이틀 17px Black, 테이블 헤더 라벨 10.5px Bold(옅은 앰버 틴트 `#FFF0CC` 배경 위), 행의 1차 식별자(이름) 11.5px Bold, 2차 필드(전화번호) 11px Regular, 3차 부가정보(주소) 11px Regular 뮤트 그레이(`#737373`). 카테고리/종류 값은 배지가 아니라 주조색(teal) 텍스트로 인라인 색상 코딩한다 — 한 행에 많은 정보를 담으면서도 위계를 유지하는 방법이다.
- **색상 사용 로직 (Color Logic in UI)**: 주조색(teal) = 주요 액션(추가/로그아웃 버튼)과 데이터 강조 텍스트(카테고리 값) · 보조색(coral) = 파괴적 액션(삭제)과 검색 실행 버튼 · 강조색(amber) = 선택된 내비게이션 상태, 추가 버튼, 테이블 헤더 틴트 배경 · 중립(흰/크림 `#F9F6F1`) = 배경과 보조 액션(수정, 필터 리셋). 상태 메시지 배너는 브랜드 3색 대신 옅은 민트 틴트(`#E5F0ED`)를 써서 알림을 절제되게 표현한다.
- **장식 모티프 (Decorative Motif)**: 이 레이아웃 단계에서 반복되는 장식은 사이드바 상단의 로고 원형 배지 하나뿐이다 — 기능 아이콘·일러스트는 아직 그리지 않았다(그래픽 크래프트는 graphic-designer 담당). pill형 카운트 배지(rounded-999)가 유일하게 반복되는 장식 형태이며, 활성 행은 흰 배지·비활성 행은 앰버 배지로 배경과 항상 반전 대비를 이룬다.
- **비고**: 색채 팔레트 자체의 스와치(Color Palette 섹션)는 이미 2px~3px 잉크 보더 + radius 16px로 만들어져 있어 위 라운드/보더 스케일과 정합적이다 — 별도 수정 불필요.

**미채택 이력**: "브랜드 컨셉 Concepts" 페이지(`34:2`)의 Concept A(Deskline, `34:3`+`44:2`)와 Concept C(Gridline, `34:5`+`46:2`)는 `❌ 미채택 —` 라벨로 보존.
