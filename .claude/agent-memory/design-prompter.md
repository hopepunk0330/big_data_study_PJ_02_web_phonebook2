  3. 순서 원칙을 별도 절로 명시(작업1→2→[3 아무때나]→4→5, 3은 1/2/4와 독립적이나 라운드 내 반드시 함께 완료).
  4. design-pl의 원본 지시 전문을 요약하지 않고 구조(목표/범위/참고자료/완료기준)로 재배열해 반환 — 다음 단계에서 이 브리프 전문이 그대로 brand-designer에게 전달될 예정이므로 세부 지시(노드ID, 색상 hex, 문서 경로, 각주 방식) 손실 없이 재구성.

### 2026-07-17 — ui-designer 브리프(단독): SCREENS 조립 — 확정 디자인 8프레임을 등록 컴포넌트 인스턴스로 재조립 (48~62차 토큰/컴포넌트 추출·검증 이후 첫 조립 라운드)
