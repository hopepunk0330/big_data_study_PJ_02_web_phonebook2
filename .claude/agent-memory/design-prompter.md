### 2026-07-17 — graphic-designer 브리프(단독): Icon/* 8종 Basic/Visual 트랙 재판정 + Basic 트랙 재드로잉(면색 제거) — "기본 아이콘 fill 금지" 신규 규칙 첫 적용
- **원본 요청**: design-pl이 이 라운드 실행을 graphic-designer로 이미 확정 전달. 사용자가 Icon/User(`96:45`)를 보고 "라인형이어도 되는 아이콘 아니냐"고 직접 지적 → 실측 결과 Icon/* 8종 + Avatar가 전부 `color/teal/500` 면색(바로 직전 2단계 라운드에서 A그룹으로 막 리바인딩 완료됨) + `color/ink/900` 3px 스트로크(B그룹, 역시 막 리바인딩 완료)로 균일하게 Visual 트랙 취급되고 있었음을 확인. `.claude/agents/graphic-designer.md`에 이미 있던 Basic/Visual 두 트랙 구분 규칙(로그아웃=Basic 예시, 카테고리/알림=Visual 예시)이 실제로는 전혀 적용되지 않고 있었던 것 — 이 규칙 문서에 "기본 아이콘은 면색을 넣지 않는다, 순수 스트로크만" 문장이 이번에 명확히 추가됐다.

(3) Basic 트랙 스트로크 두께(1.5~2px)는 `.claude/agents/graphic-designer.md` 예시값 그대로 채택하되, 색상은 새 원시값을 만들지 않고 이미 교정된 `color/ink/900`(#1a1a1a, 이번 세트 B그룹 재바인딩과 동일 계열)을 재사용하도록 지정해 일관성을 유지시켰다(다만 실제 바인딩은 design-systems 몫, 이번엔 크래프트 지시만).
